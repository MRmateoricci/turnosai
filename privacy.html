Politica de Privacidad

Ultima actualizacion: 23 de junio de 2026

1. Datos que recopilamos

Turnosai recopila nombre, telefono y mensajes de WhatsApp unicamente para gestionar reservas de turnos en los clubes que utilizan nuestro servicio.

2. Uso de datos

Los datos se usan exclusivamente para: confirmar reservas, enviar recordatorios y permitir la comunicacion entre el socio y el club.

3. WhatsApp

Utilizamos la API oficial de WhatsApp Business. Los mensajes estan cifrados. No compartimos datos con terceros.

4. Contacto

Para ejercer derechos sobre tus datos, contactanos a: vibecodevm@gmail.com
